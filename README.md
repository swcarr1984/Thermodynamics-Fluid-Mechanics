# Engineering Computation Suite

A growing collection of engineering tools, solvers, and numerical models spanning
thermodynamics, fluid mechanics, heat transfer, physics, and electrical systems.
This repository serves as a multi‑disciplinary engineering computation lab built
using Python, C++, and other languages suited for high‑performance analysis.

The goal of this project is to provide reusable, well‑documented engineering
modules that support:

- Thermodynamics calculations  
- Fluid mechanics and flow modelling  
- Heat transfer (steady, transient, convection, conduction)  
- Multi‑physics problem solving  
- Electrical and circuit analysis  
- Numerical methods and solvers  
- Engineering visualisation and plotting  
- High‑performance computation (C++ modules, Python bindings)  

---

## 📘 Project Purpose

This repository is designed as a **systems‑engineering toolkit** for solving
complex engineering problems that span multiple domains. It aims to provide:

- Clear, validated engineering formulas  
- Reusable solver functions  
- Jupyter notebooks for demonstration and exploration  
- Python and C++ implementations for performance‑critical tasks  
- A foundation for future engineering automation tools
- visual displays for results and referencing optimised problems
